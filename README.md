# ICO Service (Individual Custom cOmments)

개인 맞춤 댓글 필터링 서비스



## ICO service의 차별점
>기존보다 엄격한 필터링 서비스와 개인 맞춤 기능을 제공한다.  
#### 1. 엄격한 필터링  
 Machine Learning과 String 기반 유사도 알고리즘을 활용하여 보다 엄격한 필터링을 할 수 있도록 하였다.  
#### 2. 개인 맞춤 필터링  
 사용자로부터 키워드를 입력 받아 그를 바탕으로 필터링하는 기능을 제공한다.
